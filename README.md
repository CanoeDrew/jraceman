# jracemango

Jracemango is a rewrite of JRaceman as a web app with a golang server.

## Quick start

## Building jraceman

### Build

To compile jraceman:

    go build

### Test

To run the unit tests:

    go test ./...

### Documentation

To view the go documentation in your web browser:

    godoc -http=":6060"

Then open [localhost:6060](http://localhost:6060/) in your browser.
